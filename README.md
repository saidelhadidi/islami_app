# Islami App 🌙

A professional, elegantly designed Islamic application built with Flutter. This project is a comprehensive daily spiritual companion, focusing on **High Performance**, **Clean Code**, and an exceptional **User Experience**.

## 🚀 Project Roadmap & Progress

### ✅ Completed Milestones
- [x] **Premium Onboarding Flow:** Custom landing experience powered by `shared_preferences` to manage first-time user states.
- [x] **Holy Quran & Hadith:** A highly readable, typography-centric UI for a seamless spiritual reading experience.
- [x] **Smart Interactive Sebha:** Dynamic Zekr counter featuring **State Preservation** using `IndexedStack` (The app remembers your progress even when switching tabs!).
- [x] **Optimized Search Engine:** Instant search logic for Suras with integrated **Search History** tracking.

### ⏳ Work In Progress (Current Focus)
- [ ] **Live Quran Radio:** Integrating streaming APIs for global Quran broadcasts.
- [ ] **Real-time Prayer Times:** Implementing location-based prayer schedules via RESTful APIs.
- [ ] **Visual Azkar Cards:** 3D-styled cards for morning and evening supplications.

## 🛠️ Technical Highlights

* **Architecture:** Modular structure with a clear separation between UI components and business logic.
* **State Management:** Optimized tab switching and UI persistence using `IndexedStack`.
* **Persistence:** Local data handling using `shared_preferences` for user settings and onboarding status.
* **UI/UX:** Pixel-perfect implementation of custom designs with a focus on dark-mode aesthetics.
